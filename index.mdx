  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with the Contiguity SDK in minutes.</p>
  
  <Steps>
    <Step title="Install the package">
      Add the Contiguity SDK to your project using your preferred package manager.
      
      <CodeGroup>
      ```bash npm
      npm install contiguity
      ```
      ```bash yarn
      yarn add contiguity
      ```
      ```bash pnpm
      pnpm add contiguity
      ```
      ```bash bun
      bun add contiguity
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Initialize the client">
      Import and initialize the Contiguity client with your API token. Get your token from the [Contiguity Console](https://console.contiguity.com/).
      
      ```typescript
      import { Contiguity } from 'contiguity';
      
      const contiguity = new Contiguity('contiguity_sk_...your_token...');
      ```
    </Step>
    
    <Step title="Send your first message">
      Choose your preferred messaging channel and send a message.
      
      ```typescript
      // Send an SMS
      const response = await contiguity.text.send({
        to: "+1234567890",
        message: "Hello from Contiguity!"
      });
      
      console.log('Message sent:', response.message_id);
      ```
      
      <Note>
        You can also send emails, iMessages, WhatsApp messages, and more. Check out the [Core Features](/features/text) section for details.
      </Note>
    </Step>
  </Steps>
</div>
