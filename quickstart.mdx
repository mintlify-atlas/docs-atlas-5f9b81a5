This guide will walk you through installing the SDK, initializing a client, and sending your first message.

## Installation

<Steps>
  <Step title="Install the package">
    Choose your preferred package manager:

    <CodeGroup>
    ```bash npm
    npm install contiguity
    ```

    ```bash pnpm
    pnpm add contiguity
    ```

    ```bash yarn
    yarn add contiguity
    ```

    ```bash bun
    bun add contiguity
    ```
    </CodeGroup>
  </Step>

  <Step title="Get your API token">
    Sign up and get your API token from the [Contiguity Console](https://console.contiguity.com/). Your token will start with `contiguity_sk_`.

    <Warning>
      Keep your API token secure and never commit it to version control. Use environment variables in production.
    </Warning>
  </Step>

  <Step title="Set up environment variables (optional)">
    Create a `.env` file in your project root:

    ```bash .env
    CONTIGUITY_API_KEY=contiguity_sk_your_token_here
    ```

    Or alternatively:

    ```bash .env
    CONTIGUITY_TOKEN=contiguity_sk_your_token_here
    ```

    <Note>
      The SDK automatically reads from `CONTIGUITY_API_KEY` or `CONTIGUITY_TOKEN` environment variables if no token is provided to the constructor.
    </Note>
  </Step>
</Steps>

## Initialize the Client

Import and create a new Contiguity client instance:

<CodeGroup>
```typescript TypeScript
import { Contiguity } from 'contiguity';

// Option 1: Pass token directly
const contiguity = new Contiguity('contiguity_sk_your_token_here');

// Option 2: Use environment variable
const contiguity = new Contiguity();

// Option 3: With debug mode enabled
const contiguity = new Contiguity('contiguity_sk_your_token_here', {
  debug: true
});
```

```javascript JavaScript
const { Contiguity } = require('contiguity');

const contiguity = new Contiguity('contiguity_sk_your_token_here');
```
</CodeGroup>

<Note>
  The constructor will throw an error if:
  - No token is provided and no environment variable is set
  - The token doesn't start with `contiguity_sk_`
</Note>

## Send Your First Message

<Tabs>
  <Tab title="Text Message">
    Send a text message to any phone number:

    ```typescript
    const response = await contiguity.text.send({
      to: "+1234567890",
      message: "Hello from Contiguity!"
    });

    console.log('Message sent:', response.message_id);
    ```

    **Response:**
    ```json
    {
      "message_id": "msg_abc123...",
      "metadata": {
        "id": "req_...",
        "timestamp": "1234567890",
        "api_version": "v1",
        "object": "text_message"
      }
    }
    ```
  </Tab>

  <Tab title="Email">
    Send an email with HTML or plain text:

    ```typescript
    const response = await contiguity.email.send({
      to: "user@example.com",
      from: "Your App <no-reply@yourapp.com>",
      subject: "Welcome to Your App!",
      text: "Thanks for signing up! We're excited to have you.",
      html: "<h1>Welcome!</h1><p>Thanks for signing up!</p>"
    });

    console.log('Email sent:', response.email_id);
    ```

    **Response:**
    ```json
    {
      "success": true,
      "data": {
        "email_id": "email_xyz789..."
      }
    }
    ```
  </Tab>

  <Tab title="OTP">
    Send a one-time password for verification:

    ```typescript
    // Send OTP
    const otpResponse = await contiguity.otp.new({
      to: "+1234567890",
      language: "en",
      name: "MyApp"
    });

    console.log('OTP sent:', otpResponse.data.otp_id);

    // Later, verify the OTP entered by the user
    const verification = await contiguity.otp.verify({
      otp_id: otpResponse.data.otp_id,
      otp: "123456"
    });

    if (verification.data.verified) {
      console.log('OTP verified successfully!');
    }
    ```

    **Response:**
    ```json
    {
      "success": true,
      "data": {
        "verified": true
      }
    }
    ```
  </Tab>

  <Tab title="iMessage">
    Send an iMessage:

    ```typescript
    const response = await contiguity.imessage.send({
      to: "+1234567890",
      message: "Hello via iMessage!"
    });

    console.log('iMessage sent:', response.message_id);
    ```
  </Tab>

  <Tab title="WhatsApp">
    Send a WhatsApp message:

    ```typescript
    const response = await contiguity.whatsapp.send({
      to: "+1234567890",
      message: "Hello via WhatsApp!"
    });

    console.log('WhatsApp message sent:', response.message_id);
    ```
  </Tab>
</Tabs>

## Error Handling

All SDK methods can throw errors. Use try-catch blocks to handle them:

```typescript
import { Contiguity, ContiguityError } from 'contiguity';

const contiguity = new Contiguity();

try {
  const response = await contiguity.text.send({
    to: "+1234567890",
    message: "Hello!"
  });
  console.log('Success:', response.data);
} catch (error) {
  if (error instanceof ContiguityError) {
    console.error('API Error:', error.message);
    console.error('Status:', error.status);
  } else {
    console.error('Unexpected error:', error);
  }
}
```
